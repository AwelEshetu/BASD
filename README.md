# BASD
To run the app locally:
 1. create virtual environment 
  virtualenv -p python3 venv
  
 2. activate your virtual env
  source ./venv/bin/activate
  
 3. install dependecies from requirements
   pip install -r requirements.txt
 
 4. run the app
    python app.py
